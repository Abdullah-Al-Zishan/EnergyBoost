Some tools for running scripts¶

Run local batch¶

  • Run all scenarios given home id will run all scenarios of homes given home id.
  • Run all scenarios give filder name will run all scenarios of all homes inside one given folder.

Run on Compute Canada jobs¶

  • Run scripts on Compute Canada solver when running simnulations on compute canada server like Graham, hearder file for assign compute resources for one job need to be added.

For more details about schedule jobs on compute canada, please refer to [running jobs on Compute Canada <https://docs.computecanada.ca/wiki/Running_jobs>`_

  • List of all home id generate list of all home id which will be used to create seprate jobs for different homes on server.
  • Create shell scripts for one given home will create shell scripts to run scenarios for all given home on server.
  • Run indiviual jobs for all homes will create one single jobs for all homes so that mutiple jobs can be run simultaneously on Compute Canada server.
